Settings
===========

.. setting:: EMBED_VIDEO_BACKENDS

EMBED_VIDEO_BACKENDS
--------------------

List of backends to use.

Default:: 
    
  EMBED_VIDEO_BACKENDS = (
      'embed_video.backends.YoutubeBackend',
      'embed_video.backends.VimeoBackend',
      'embed_video.backends.SoundCloudBackend',
  )
